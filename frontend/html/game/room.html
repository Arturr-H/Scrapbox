How to play

The aim of the game is to form sentences and answers to questions using words that the players in the game have written before the game starts.

The game starts with everyone writing a short story - this is where all the words come from when playing the game. All the words are shuffled together and you have to answer the following questions in the game with the words. Voting takes place after everyone has answered their questions. Then you have to vote for the card with the funniest sentence, the one with the most votes at the end of the game wins.
